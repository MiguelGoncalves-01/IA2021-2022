-----Para correr o programa do MINIMAX-----
Compilar: javac ConnectFour.java
Correr: java ConnectFour

Passos do programa:
1: Escolher dificuldade (escolher a 4 para se criar a árvore de pesquisa toda)
2: Escolher o jogador inicial
3: Para cada jogada escolher um número de [0-6]
